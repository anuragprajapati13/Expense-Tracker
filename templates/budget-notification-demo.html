🔧 Usage Examples

Basic Usage:

budgetNotificationManager.show( 50000, // Amount spent 40000, // Budget limit { message: '⚠️ Budget Exceeded! You are spending too much', suggestion: 'Try reducing food or shopping expenses', autoHide: true, autoHideDelay: 8000 } );

With Custom Callbacks:

budgetNotificationManager.show( 75000, 50000, { message: '⚠️ Budget Exceeded!', suggestion: 'You are 50% over budget', onReviewClick: () => { console.log('Review clicked'); window.location.href = '/reports'; }, onBudgetClick: () => { console.log('Budget clicked'); // Open budget settings } } );

Quick Check & Show:

// Only show if budget is exceeded budgetNotificationManager.checkAndShow(55000, 50000, { message: '⚠️ Budget Exceeded!', suggestion: 'Your expenses have exceeded the budget' });

Close Notification:

budgetNotificationManager.close();

✨ Features

  • ✅ Smooth slide-in animation from top-right
  • ✅ Pulsing glow effect on warning icon
  • ✅ Floating animation for visual appeal
  • ✅ Glassmorphism design with blur effects
  • ✅ Responsive layout (works on mobile & desktop)
  • ✅ Light & Dark theme support
  • ✅ Progress bar showing % budget used
  • ✅ Auto-hide after 8 seconds (configurable)
  • ✅ Close button with smooth exit animation
  • ✅ Custom callbacks for button actions
  • ✅ Real-time budget calculations

🎨 Customization

All notification text, timing, and actions are fully customizable. The component adapts to both light and dark themes automatically based on user preference.
