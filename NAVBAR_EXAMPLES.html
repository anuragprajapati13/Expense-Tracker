🚀 Interactive Navbar - Usage Examples

📬 Add Notifications

Add notifications to the navbar in real-time:

navbarFunctions.addNotification(   'Transaction Added',   '₹2,500 for Groceries',   'success' );
Click buttons above to add notifications →

🔔 Update Notification Count

Update the notification badge count:

navbarFunctions.updateNotificationCount(5);
Click buttons to update notification count →

🌓 Toggle Theme

Switch between dark and light themes programmatically:

navbarFunctions.toggleTheme();
Current theme: Dark

📋 Toggle Dropdowns

Open/close dropdowns programmatically:

navbarFunctions.toggleDropdown('notificationsDropdown'); navbarFunctions.toggleDropdown('userDropdown');

⌨️ Keyboard Shortcuts

  • Ctrl+K / Cmd+K: Focus search input
  • Escape: Close open dropdowns

✨ Features

  • Dark neon glassmorphism UI design
  • Fully interactive search with suggestions
  • Real-time notifications with badge count
  • User dropdown with profile options
  • Theme toggle with auto-save
  • Quick stats display (balance, daily expenses)
  • CSS keyframe animations throughout
  • GSAP integration for enhanced animations
  • Fully responsive design
  • Mobile hamburger menu

🎨 Notification Types

The navbar supports three notification types, each with unique styling:

  • 'success' - Green icon (✓ Check mark)
  • 'warning' - Orange icon (⚠ Warning)
  • 'info' - Blue icon (ℹ Information)

📖 API Reference

  • navbarFunctions.addNotification(title, message, type, time)
  • navbarFunctions.updateNotificationCount(count)
  • navbarFunctions.toggleTheme()
  • navbarFunctions.toggleDropdown(dropdownId)
